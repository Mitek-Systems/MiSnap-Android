class CameraView @JvmOverloads constructor(context: Context, attrs: AttributeSet? = null, @AttrRes defStyleAttr: Int = 0, @StyleRes defStyleRes: Int = 0) : SurfaceView

The CameraView provides a straightforward way to create and manage a camera. On creation, the view will display the camera's preview and provide access to preview frames through the previewFrames property.
