ReviewCondition

enum ReviewCondition : Enum<DocumentAnalysisFragment.ReviewCondition>

Sets the condition for when to navigate to the document review action. Supported conditions are:

  • ALWAYS Always navigate to the document review action.

  • NEVER Never navigate to the document review action.

  • MANUAL Only navigate to the document review action when the image was taken manually.

  • WARNINGS Only navigate to the document review action if there were warnings associated with the image.
